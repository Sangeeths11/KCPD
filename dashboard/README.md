# Streamlit App Setup with Conda

This guide will help you set up a conda environment and run the Streamlit app.

## Prerequisites

- Conda (Anaconda or Miniconda)
- Python 3.x
- `requirements.txt` file

## Steps

### 1. Create a Conda Environment

Navigate to your project directory in the terminal and create a conda environment using the following command:

```bash
conda create --name myenv python=3.x
```

```bash
pip install -r requirements.txt
```

### 2. Start streamlit app
```bash
streamlit run app.py
```
